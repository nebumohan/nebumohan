## 🚀 Featured Project

### 🔹 AI-Powered Vulnerability Scanner

👉 https://github.com/nebumohan/ai-vulnerability-scanner

A custom SAST tool that combines traditional static analysis with AI-driven insights.

**What it does:**

* Detects vulnerabilities using Bandit and Semgrep
* Identifies issues like command injection and unsafe code execution
* Uses LLM (Gemini API) to explain risks and suggest fixes
* Generates structured HTML reports and stores findings in SQLite

**Why it matters:**
Traditional scanners generate large reports that are hard to interpret.
This tool focuses on **making vulnerabilities understandable and actionable for developers**.
